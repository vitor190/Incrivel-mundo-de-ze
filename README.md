# 🎓 Incrível Mundo de Zé

Um jogo 2D de exploração desenvolvido em **Godot Engine** como projeto da disciplina de Computação Gráfica da UNIFOR.

## 📖 Sobre o jogo

Acompanhe a vida de **Zé**, um estudante universitário que percorre dia a dia os blocos da UNIFOR em busca de crescimento acadêmico e pessoal. Explore o campus, interaja com o ambiente e ajude Zé a evoluir na sua jornada como estudante.

## 🕹️ Funcionalidades

- Personagem jogável com animações
- Mapa do campus da UNIFOR
- Minimapa para navegação
- Trilha sonora original em loop

## 🛠️ Tecnologias

- [Godot Engine 4](https://godotengine.org/)
- GDScript

## 🚀 Como rodar

1. Clone o repositório:
   ```bash
   git clone https://github.com/vitor190/Incrivel-mundo-de-ze.git
   ```
2. Abra o Godot Engine
3. Importe a pasta do projeto
4. Rode com **F5**
